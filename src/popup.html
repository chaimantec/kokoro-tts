Sherlock TTS Engine

This Sherlock-themed extension provides a custom TTS engine that uses a local API endpoint for speech synthesis.

Status: Checking...
Model Status: Unknown

How to use:

  1. Enter or paste text in the box above and click "Play Text", or
  2. Select text on any webpage and either:
    • Right-click on the selected text and choose "Read with Sherlock TTS" from the context menu, or
    • Press Ctrl+Shift+S (⌘+Shift+S on Mac)
